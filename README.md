# AI Adoption & HEART Metrics Analytics Platform

## Overview
This project builds an enterprise-grade analytics system to measure AI product adoption using the HEART framework:
Happiness, Engagement, Adoption, Retention, and Task Success.

It simulates enterprise AI usage behavior and generates actionable insights for AI adoption strategy.

---

## Features
- HEART metrics computation across business divisions
- AI maturity scoring model
- Cohort segmentation (Power Users vs Normal Users)
- Token usage and adoption analytics
- Power BI-ready datasets

---

## Tech Stack
- Python (Pandas, NumPy, Scikit-learn)
- SQL (data modeling logic)
- Power BI (dashboarding)
- Data Simulation for enterprise AI logs

---

## Project Structure
- AI usage simulation dataset
- HEART metrics computation
- Cohort analysis engine
- AI maturity scoring model
- Exported datasets for BI dashboards

---

## Key Insights
- Identifies high vs low AI adoption divisions
- Measures AI usage efficiency across teams
- Detects power users and engagement clusters
- Provides executive-level AI adoption visibility
